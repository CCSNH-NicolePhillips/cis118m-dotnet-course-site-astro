# Week 12 Homework: Mini Test Harness

## Overview
Write 2–3 pure methods and a tiny test runner that checks expected vs actual results.

## Requirements checklist (acceptance criteria)
- [ ] Includes at least 2 methods that return values (no reading input inside them)
- [ ] Defines at least 6 test cases total (inputs + expected outputs)
- [ ] Runs the test cases and prints pass/fail for each
- [ ] Prints a final summary: total passed / total tests
- [ ] Runs without errors
