# Week 12 Extra Practice: Test case gym

Pick **1–2** items (optional).

- Pick a method you wrote earlier (Add, IsEven, IsBetween). Write 8 test cases, including edge cases.
- Write a method `SafeDivide(double a, double b)` that returns 0 when b is 0 (or returns null/NaN if you prefer). Write test cases.
- Challenge: Create a `RunTests()` method that prints pass/fail for 5 test cases.
