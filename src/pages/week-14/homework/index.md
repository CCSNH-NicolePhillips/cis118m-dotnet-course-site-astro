# Week 14 Homework: Budget Tracker (encapsulation)

## Overview
Create a simple Budget class where spending can’t exceed the current balance (unless you allow overdraft—your choice, but be clear).

## Requirements checklist (acceptance criteria)
- [ ] Defines a `Budget` class with Balance
- [ ] Deposit adds money; Spend attempts to subtract money
- [ ] Spend returns bool (true if successful)
- [ ] Main program demonstrates at least 3 spends (at least one fails)
- [ ] Output is labeled and easy to read
- [ ] Runs without errors
