# Week 10 Lab: Student Profile Class

## Overview
Create a `Student` class with properties and a method that returns a formatted summary string.

## Requirements checklist (acceptance criteria)
- [ ] Defines a `Student` class with at least 3 properties (example: Id, Name, Major)
- [ ] Includes a constructor that sets at least 2 properties
- [ ] Includes a method (example: `Summary()`) that returns a string (no printing inside the method)
- [ ] Creates at least 2 Student objects in main and prints their summaries
- [ ] Runs without errors
