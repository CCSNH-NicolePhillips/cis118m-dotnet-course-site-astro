# Week 10 Homework: Bank Account (tiny simulation)

## Overview
Create a `BankAccount` class and simulate deposits/withdrawals with simple rules.

## Requirements checklist (acceptance criteria)
- [ ] Defines a `BankAccount` class with Owner and Balance
- [ ] Constructor sets Owner and starting Balance
- [ ] Includes `Deposit(double amount)` and `Withdraw(double amount)` methods
- [ ] Withdraw rejects invalid amounts (like negative or more than balance) with a friendly message or a return value
- [ ] Main program demonstrates at least 2 deposits and 1 withdrawal
- [ ] Output is labeled and program runs without errors
