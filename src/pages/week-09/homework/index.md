# Week 9 Homework: Password Strength Checker (simple)

## Overview
Create a password checker that scores a password using basic rules (no need for ‘real-world security’ yet).

## Requirements checklist (acceptance criteria)
- [ ] Prompts user for a password (string)
- [ ] Checks at least 4 rules (example: length >= 8, contains a digit, contains uppercase, contains lowercase)
- [ ] Prints a score or category (Weak / Okay / Strong) based on how many rules passed
- [ ] Prints at least one helpful improvement tip when weak
- [ ] Runs without errors
