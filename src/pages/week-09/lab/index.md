# Week 9 Lab: Username Validator

## Overview
Build a program that checks whether a username follows a few simple rules and prints clear feedback.

## Requirements checklist (acceptance criteria)
- [ ] Prompts the user for a username
- [ ] Trims leading/trailing spaces
- [ ] Checks at least 3 rules (example: length 4–12, no spaces, contains a letter)
- [ ] Prints a friendly message for each failed rule OR a single summary message
- [ ] Accepts usernames case-insensitively (your rules should define this clearly)
- [ ] Runs without errors

## Submit to Canvas
- Submit code (paste/export)
- Submit screenshot of output showing one invalid username and one valid username
