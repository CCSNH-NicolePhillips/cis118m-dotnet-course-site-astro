# Week 16 Lab: Final Project Submission

## Overview
Finish and submit your final project. Show that it works and that you tested it.

## Requirements checklist (acceptance criteria)
- [ ] Implements at least 4 MVP features from your proposal
- [ ] Uses at least one class and at least one List of objects
- [ ] Handles invalid input without crashing
- [ ] If using files: loads if file exists and saves correctly
- [ ] Includes clear labeled output for grading
- [ ] Includes at least 2 screenshots showing key features
- [ ] Includes a short reflection (3–5 sentences)

## Submit to Canvas
- Submit your code (zip or paste)
- Submit screenshots
- Submit reflection text
