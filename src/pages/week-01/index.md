# Week 1 Overview

**Focus:** Getting started with .NET and C# — everything happens on our course site!

**Total time:** 210 minutes (3.5 hours)

## Your Week 1 Checklist

Complete these items in order. Each one builds on the last:

| Item | Time | What You'll Do | Link |
|------|------|----------------|------|
| **Start Here + Site Tour** | 30 min | Learn how to use the course site, IDE, and get help | [Start Here](./start-here/) |
| **Syllabus + Acknowledgement** | 30 min | Read the syllabus and complete the acknowledgement quiz | [Syllabus](./syllabus/) → [Ack Quiz](./syllabus-ack/) |
| **Lesson 1** | 35 min | What is .NET vs C#? Write your first Hello World program | [Lesson 1](./lesson-1/) |
| **Lesson 2** | 35 min | Console output, variables, and strings | [Lesson 2](./lesson-2/) |
| **Extra Practice** | 20 min | Mini drills to reinforce what you learned | [Extra Practice](./extra-practice/) |
| **Checkpoint Quiz** | 20 min | Test your understanding (auto-graded, on-site) | [Checkpoint Quiz](./checkpoint-quiz/) |
| **Lab 1** | 40 min | Build an "About Me" console app (graded assignment) | [Lab 1](./lab-1/) |

**Total:** 210 minutes

## Deliverables (What Gets Graded)

By the end of Week 1, you must submit:

1. ✅ **Syllabus Acknowledgement** — Complete the quiz (shows you read the syllabus)
2. ✅ **Checkpoint Quiz** — Score 70% or higher (you can retake)
3. ✅ **Lab 1** — Submit your "About Me" console app (must run + meet requirements)

## Learning Objectives

By the end of this week, you'll be able to:

- Explain (in simple words) what .NET and C# are
- Run a C# program and read the output
- Create and use variables (strings and numbers)
- Print formatted output with labels
- Use string concatenation and interpolation
- Read error messages and fix simple bugs
