**Time: 45 minutes**

## Tips

**Stuck?** Here's a hint:

```csharp
string name = "Nikki";
string course = "CIS 118M";
Console.WriteLine($"My name is {name}.");
Console.WriteLine($"I'm taking {course}.");
```

**Common mistakes to avoid:**
- Forgetting quotes around string values
- Forgetting semicolons at the end of statements
- Forgetting the `$` in front of interpolated strings
- Misspelling `Console.WriteLine`
