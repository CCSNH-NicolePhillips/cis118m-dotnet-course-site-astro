# Lab 1: Welcome Program

**Time: 45 minutes**  
**Points: 100**

## Objective

Create a console program that prints a welcome message with information about yourself and this course.

---

## Requirements

Your program must:

1. ✅ Print **exactly 4 lines** of output
2. ✅ Include a **header comment** with your name and the assignment name
3. ✅ Use **at least 2 variables** (string, int, or both)
4. ✅ Use **either** string interpolation **or** concatenation (your choice)
5. ✅ Print the following information:
   - Your name
   - The course name or number (e.g., "CIS 118M")
   - Your goal for this course
   - A fun fact about yourself

---

## Example Output

```
Welcome to CIS 118M!
My name is Nikki Smith.
My goal this term is to build my first real app.
Fun fact: I love hiking and photography.
```

Your output can be different, but it should follow the same structure.

## Tips

**Stuck?** Here's a hint:

```csharp
string name = "Nikki";
string course = "CIS 118M";
Console.WriteLine($"My name is {name}.");
Console.WriteLine($"I'm taking {course}.");
```

**Common mistakes to avoid:**
- Forgetting quotes around string values
- Forgetting semicolons at the end of statements
- Forgetting the `$` in front of interpolated strings
- Misspelling `Console.WriteLine`
