# Week 1 Homework: About-Me Console Card

## Overview
Write a console program that prints a short, readable ‘card’ about you (5–7 lines).

## Requirements checklist (acceptance criteria)
- [ ] Prints 5–7 lines
- [ ] Includes your name (or preferred name)
- [ ] Includes why you’re taking this class
- [ ] Includes a favorite app/game/tech you use
- [ ] Includes one fun fact
- [ ] Includes one line that starts with: “This semester I want to learn…”
- [ ] Runs without errors and is easy to read
