# Intro to .NET (CIS118M)

Welcome. This site is the **Read / Review** hub linked from Canvas.

## How to use this course site

- Start in Canvas for the weekly checklist.
- Use the matching week here for the lesson pages and examples.
- Use the “Open Browser Code Editor” link to practice code without installing anything.

## How to use the editor

- Open the browser editor: [/editor/?week=01](/editor/?week=01)
- Pick the week starter from the dropdown (Week 1 is preselected).
- Type or paste your code; it auto-saves per week/starter.
- Use Reset to restore the starter, Copy to copy, Download to save `WeekXX.cs`.
- Run currently uses a safe stub; you can copy code to run locally if needed.
