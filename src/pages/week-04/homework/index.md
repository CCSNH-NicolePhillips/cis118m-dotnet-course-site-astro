# Week 4 Homework: Menu Choice Program

## Overview
Create a simple menu (1–3 options). Use if/else to respond to the user’s choice.

## Requirements checklist (acceptance criteria)
- [ ] Prints a menu with at least 3 options
- [ ] Reads the user choice (as text or number)
- [ ] Handles invalid choices with a friendly message
- [ ] Uses if/else or else-if (or switch if you want, but if/else is fine)
- [ ] Runs without errors

## Submit to Canvas
- Submit code (paste/export)
- Submit screenshot of output showing one valid choice and one invalid choice
