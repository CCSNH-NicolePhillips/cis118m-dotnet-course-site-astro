# Lab 2: System Status Report

**Time: 45 minutes**  
**Points: 100**

## Objective

Create a well-structured console program that simulates a "System Status Report" - demonstrating your understanding of **namespaces**, **class structure**, **the Main method**, and **professional documentation**.

---

## ðŸŽ¯ The Mission

You're building a diagnostic tool that outputs a formatted system status report. This lab focuses on **program structure** - not variables or math. You'll practice:

- Organizing code with `namespace` and `class`
- Using the correct `Main` method signature
- Writing XML documentation comments (`///`)
- Returning an exit code from `Main`

---

## Requirements

Your program must include:

### 1. âœ… Header Comments
- Line 1: Your name (e.g., `// Name: Jane Doe`)
- Line 2: Assignment name (e.g., `// Assignment: Lab 2 - System Status Report`)

### 2. âœ… Namespace & Class Structure
- Wrap your code in a `namespace` called `SystemDiagnostics`
- Create a `class` called `StatusReport`

### 3. âœ… XML Documentation
- Add an XML `<summary>` comment above your `Main` method explaining what it does

### 4. âœ… Main Method with Exit Code
- Use `static int Main()` instead of `static void Main()`
- Return `0` at the end to indicate success

### 5. âœ… Output (at least 5 lines)
Print a formatted status report with these sections:
- A header line (e.g., `========== SYSTEM STATUS REPORT ==========`)
- System name
- Status (e.g., "All systems operational")
- A separator line
- A footer with your name as the "technician"

---

## Example Output

```
========================================
       SYSTEM STATUS REPORT
========================================
System: CIS118M Learning Platform
Status: All systems operational
Uptime: Online since Week 1
----------------------------------------
Technician: Jane Doe
Report Complete. Exit Code: 0
========================================
```

Your output can be different, but it should be **formatted** and **professional**.

```csharp
        /// Generates and displays a system status report.
        /// </summary>
        static int Main()
        {
            // Your Console.WriteLine statements go here
            
            // Return 0 to indicate successful execution
            return 0;
        }
    }
}
```

## Steps

1. **Update the header comments** with your name and assignment
2. **Keep the namespace** `SystemDiagnostics` and class `StatusReport`
3. **Add your XML documentation** above `Main` (already started for you)
4. **Write at least 5 `Console.WriteLine` statements** to create your report
5. **Make sure `Main` returns `int`** and ends with `return 0;`
6. **Test your code** by clicking Run
7. **Submit** when your output looks professional

## Tips

**Stuck?** Here are some hints:

```csharp
Console.WriteLine("========================================");
Console.WriteLine("       SYSTEM STATUS REPORT");
Console.WriteLine("========================================");
Console.WriteLine("System: Your System Name Here");
```

**Common mistakes to avoid:**
- Forgetting to change `void` to `int` in the Main signature
- Forgetting the `return 0;` statement
- Missing the namespace or class wrapper
- Forgetting quotes around your text
