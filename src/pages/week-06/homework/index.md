# Week 6 Homework: Utility Methods Pack

## Overview
Create a small set of utility methods (mini toolbelt). Then demonstrate them in the main program.

## Requirements checklist (acceptance criteria)
- [ ] Includes at least 4 methods you wrote
- [ ] At least 2 methods return values
- [ ] At least 1 method accepts parameters
- [ ] Main program calls every method at least once
- [ ] Output is labeled and easy to read
- [ ] Runs without errors
