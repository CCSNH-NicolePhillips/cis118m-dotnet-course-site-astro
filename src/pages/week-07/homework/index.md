# Week 7 Homework: Search + Count Tasks

## Overview
Use arrays to search for values and count things (like how many match a rule).

## Requirements checklist (acceptance criteria)
- [ ] Uses at least one array (int[] or string[])
- [ ] Implements at least 2 counting tasks (example: count > 50, count even, count names starting with A)
- [ ] Implements at least 1 searching task (example: does the array contain X?)
- [ ] Prints labeled results
- [ ] Runs without errors
