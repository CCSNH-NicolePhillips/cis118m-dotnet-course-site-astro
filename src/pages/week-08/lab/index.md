# Week 8 Lab: Simple To-Do List App

## Overview
Create a tiny to-do list program. You can keep it simple: add a few tasks, remove one, print the list.

## Requirements checklist (acceptance criteria)
- [ ] Uses a `List<string>`
- [ ] Adds at least 3 items
- [ ] Removes at least 1 item (by value or index)
- [ ] Prints all items (foreach recommended)
- [ ] Prints the count of tasks
- [ ] Runs without errors
