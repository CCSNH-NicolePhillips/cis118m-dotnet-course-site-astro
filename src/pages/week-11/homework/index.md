# Week 11 Homework: Inventory Tracker (objects + list)

## Overview
Create an `Item` class (Name, Quantity, maybe Price). Store items in a list and support at least one search or update.

## Requirements checklist (acceptance criteria)
- [ ] Defines an `Item` class with at least 2 properties
- [ ] Creates a `List<Item>` with at least 4 items
- [ ] Prints all items in a readable format
- [ ] Allows updating an item (example: change quantity) OR searching by name
- [ ] Handles not found cases
- [ ] Runs without errors
