# Week 11 Lab: Class Roster Manager

## Overview
Build a tiny roster app: store students, print the roster, and search by ID.

## Requirements checklist (acceptance criteria)
- [ ] Uses a `Student` class (or similar) with at least 2 properties
- [ ] Stores at least 3 objects in a `List<Student>`
- [ ] Prints the roster using foreach
- [ ] Asks the user for an ID and searches for it
- [ ] Prints ‘found’ or ‘not found’ message clearly
- [ ] Runs without errors
