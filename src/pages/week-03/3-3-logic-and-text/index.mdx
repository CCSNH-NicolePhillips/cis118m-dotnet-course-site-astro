## ‚úÖ bool: The Decision Type

A `bool` (Boolean) stores only two possible values: `true` or `false`. 

Named after mathematician George Boole, this type is the foundation of all computer logic.

```csharp
bool isSystemActive = true;
bool hasPermission = false;
bool isLoggedIn = true;
```

### Common Uses for bool:
- **System states:** Is the server online? Is the user authenticated?
- **Flags:** Should we enable this feature? Has the task completed?
- **Conditions:** Results of comparisons (covered in Week 06)

<TryItNowRunner 
  starterId="week-03-bool-demo"

  code={`using System;

    static void Main()
    {
        bool isEngineRunning = true;
        bool isMaintenanceMode = false;
        
        Console.WriteLine("Engine Running: " + isEngineRunning);
        Console.WriteLine("Maintenance Mode: " + isMaintenanceMode);
        
        // Toggle the state
        isMaintenanceMode = true;
        Console.WriteLine("After toggle: " + isMaintenanceMode);
    }
}`}

### Default Value

If you declare a `bool` without initializing it (in a class field), its default value is `false`.

```csharp
bool isVerified;  // Default: false
```

## üî§ char: The Single Character

A `char` stores exactly **one** Unicode character. Use single quotes!

```csharp
char grade = 'A';
char initial = 'J';
char symbol = '#';
char newline = '\n';  // Special escape character
```

### char vs string:
- `char` = Single quotes: `'A'`
- `string` = Double quotes: `"A"` (even for one character)

<TryItNowRunner 
  starterId="week-03-char-demo" 
  height={300}
  code={`using System;

        // Each char is a single character
        char first = 'C';
        char second = '#';
        Console.WriteLine("Language: " + first + second);
    }
}`}
  instructions="Run this code. Notice how chars combine when concatenated!"

## üîí String Immutability

<div class="critical-concept">
  <h3>‚ö° Key Concept: Strings are Immutable</h3>
  <p>
    Once a string is created in memory, it <strong>cannot be changed</strong>. 
    Any operation that appears to modify a string actually creates a <strong>new string</strong>.
  </p>
</div>

### What Immutability Means

```csharp
string message = "Hello";
message = message + " World";  // Creates a NEW string in memory
```

## üîó String Concatenation

You can combine strings using the `+` operator:

```csharp
string first = "Hello";
string second = "World";
string combined = first + " " + second;  // "Hello World"
```

### String Interpolation (Modern C#)

A cleaner way to build strings:

<TryItNowRunner 
  starterId="week-03-interpolation" 
  height={320}
  code={`using System;

class Program
{
    static void Main()
    {
        string name = "Engineer";
        int level = 5;
        
        // Old way (concatenation)
        Console.WriteLine("Welcome, " + name + "! Level: " + level);
        
        // Modern way (interpolation with $)
        Console.WriteLine($"Welcome, {name}! Level: {level}");
    }
}`}
  instructions="The $ before the string allows you to embed variables in {curly braces}!" 
/>

## üìä Type Summary

| Type | Size | Values | Literal Syntax | Use Case |
|------|------|--------|----------------|----------|
| `bool` | 1 bit* | `true` / `false` | `true` | System states, flags |
| `char` | 16-bit | Single Unicode char | `'A'` | Single characters, grades |
| `string` | Variable | Text sequence | `"Hello"` | Names, messages, data |

*bool uses at least 1 byte due to memory alignment
