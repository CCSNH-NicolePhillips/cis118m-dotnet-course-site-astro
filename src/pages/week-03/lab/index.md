# Week 3 Lab: Input + Math (with validation)

## Overview
Read a number from the user, validate it, and do a simple calculation.

## Requirements checklist (acceptance criteria)
- [ ] Prompts the user clearly
- [ ] Uses TryParse (int or double) to validate input
- [ ] Re-prompts at least once when input is invalid
- [ ] Performs at least one calculation using the validated number
- [ ] Runs without errors
