## üí° Hints

<details>
<summary>Hint 1: Declaring a decimal</summary>

```csharp
decimal serverCost = 45.99m;  // The 'm' suffix is required!
```

Without the `m`, C# treats 45.99 as a `double` and won't let you assign it to a `decimal`.

</details>

<details>
<summary>Hint 2: Displaying booleans nicely</summary>

```csharp
bool isActive = true;

// Instead of printing "True" or "False":
string status = isActive ? "Active" : "Inactive";
Console.WriteLine("Status: " + status);
```

This uses a ternary operator (covered later) to show "Active" instead of "True".

</details>

<details>
<summary>Hint 3: Formatting currency</summary>

```csharp
decimal cost = 45.99m;
Console.WriteLine("Cost: $" + cost);
// Or using string interpolation:
Console.WriteLine($"Cost: ${cost}");
```

</details>
