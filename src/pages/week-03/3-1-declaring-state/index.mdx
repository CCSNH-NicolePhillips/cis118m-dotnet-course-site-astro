## üèóÔ∏è What is a Variable?

A **variable** is a named storage location in your computer's memory (RAM). Think of it as a labeled box that holds a single piece of data.

<div class="concept-highlight">
  <h3>The Parking Lot Analogy</h3>
  <p>Imagine your computer's memory as a giant parking garage:</p>
  <ul>
    <li><strong>Variable Name</strong> = The parking spot number (e.g., "A-42")</li>
    <li><strong>Variable Type</strong> = What kind of vehicle can park there (car, motorcycle, truck)</li>
    <li><strong>Variable Value</strong> = The actual vehicle parked in that spot</li>
  </ul>
</div>

## üìù Declaration vs. Initialization

There are two distinct steps when creating a variable:

        <pre><code>int systemPort;  // Reserved, but empty</code></pre>
        <p><strong>‚ö†Ô∏è Warning:</strong> Using a variable before giving it a value causes a compiler error!</p>

        <p><strong>Initialization</strong> assigns the first value to a variable.</p>
        <pre><code>systemPort = 8080;  // Now it has a value</code></pre>
        <p>The variable is now ready to use in your program.</p>
      `
    },
    {

        <p><strong>‚úÖ Recommended:</strong> This prevents accidental use of uninitialized variables.</p>

## üîç Anatomy of a Variable Declaration

```csharp
int  userCount  =  1500;
‚îÇ       ‚îÇ       ‚îÇ    ‚îÇ
‚îÇ       ‚îÇ       ‚îÇ    ‚îî‚îÄ‚îÄ Value (the data stored)
‚îÇ       ‚îÇ       ‚îî‚îÄ‚îÄ Assignment operator
‚îÇ       ‚îî‚îÄ‚îÄ Identifier (the name)
‚îî‚îÄ‚îÄ Type (what kind of data)
```

## üìõ Naming Rules (Identifiers)

C# has strict rules about what you can name your variables:

<div class="rules-grid">
  <div class="rule valid">
    <h4>‚úÖ Valid Names</h4>
    <ul>
      <li><code>userCount</code></li>
      <li><code>firstName</code></li>
      <li><code>_privateField</code></li>
      <li><code>item1</code></li>
      <li><code>MAX_SIZE</code></li>
    </ul>
  </div>
  <div class="rule invalid">
    <h4>‚ùå Invalid Names</h4>
    <ul>
      <li><code>1stPlace</code> ‚Äî Can't start with number</li>
      <li><code>user-count</code> ‚Äî No hyphens</li>
      <li><code>first name</code> ‚Äî No spaces</li>
      <li><code>class</code> ‚Äî Reserved keyword</li>
      <li><code>int</code> ‚Äî Reserved keyword</li>
    </ul>
  </div>
</div>

## üîÑ Reassignment

Once a variable is declared, you can change its value (unless it's a constant):

<TryItNowRunner 
  starterId="week-03-reassignment" 
  height={340}
  code={`using System;

class Program
{
    static void Main()
    {
        int score = 0;
        Console.WriteLine("Starting score: " + score);
        
        score = 100;  // Reassignment
        Console.WriteLine("After bonus: " + score);
        
        score = score + 50;  // Using current value
        Console.WriteLine("After quest: " + score);
    }
}`}
  instructions="Watch how the score variable changes. Try adding another reassignment at the end."
