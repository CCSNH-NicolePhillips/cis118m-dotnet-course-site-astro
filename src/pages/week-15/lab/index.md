# Week 15 Lab: Final Project Proposal

## Overview
Create a short proposal for your final project: what it is, features, and what ‘done’ means.

## Requirements checklist (acceptance criteria)
- [ ] Project title + one-sentence description
- [ ] List 4–6 MVP features
- [ ] Each feature has 2–3 acceptance criteria checkboxes
- [ ] Identify what data you will store (objects + list)
- [ ] Decide if you will save to a file (recommended) and what file format

## Submit to Canvas
- Submit your proposal text (copy/paste) OR a screenshot of it
