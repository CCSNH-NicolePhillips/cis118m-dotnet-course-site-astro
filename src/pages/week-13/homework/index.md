# Week 13 Homework: CSV Roster Saver

## Overview
Create a small roster app that saves students to a CSV file and loads them back.

## Requirements checklist (acceptance criteria)
- [ ] Defines a Student class with Id and Name (and optional Major)
- [ ] Stores students in a `List<Student>`
- [ ] Saves to `students.csv` (one student per line)
- [ ] Loads from file if it exists at startup
- [ ] Prints the roster in a readable format
- [ ] Runs without errors
